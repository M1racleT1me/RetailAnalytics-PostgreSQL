# RetailAnalytics — Customer Segmentation

Аналитическая система для розничной сети: сегментация клиентов, расчёт метрик.

## Цель проекта
Построить базу данных, которая:
- анализирует клиентов по среднему чеку, частоте покупок и churn;
- формирует персональные скидки по трём стратегиям;
- реализует витрины данных и ролевую модель PostgreSQL.

## Технологии
PostgreSQL 14+, PL/pgSQL, SQL views, оконные функции, агрегаты, CTE.
